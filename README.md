# StatsfootPRO


StatsFoot es una aplicación móvil desarrollada en Flutter con backend en Supabase, pensada para organizar partidos locales de fútbol, gestionar equipos y registrar estadísticas de jugadores en tiempo real.

- Descripción

  StatsFoot permite crear y unirse a partidos de fútbol locales, registrar resultados y seguir el rendimiento individual de cada jugador. Está diseñada para grupos de amigos o ligas amateur que buscan llevar un control más profesional de sus partidos sin complicaciones.
  
  La aplicación incorpora además un sistema para registrar goles y asistencias mediante una pulsera conectada, lo que permite registrar eventos del partido sin necesidad de usar el móvil durante el juego.

- Características principales

    - Organización de partidos: creación y gestión sencilla de partidos locales.
  
       - Gestión de equipos: creación de equipos, invitaciones y seguimiento del rendimiento de los jugadores.
    
       - Registro de goles y asistencias: mediante una pulsera conectada por Bluetooth o NFC (en desarrollo).
    
    - Estadísticas detalladas: control de goles, asistencias, victorias y rendimiento general.

    - Autenticación segura: login y registro gestionados con Supabase Auth.
  
    - Sincronización en la nube: datos almacenados y actualizados automáticamente en Supabase.
  
    - Tecnologías utilizadas
  
        Frontend: Flutter
        
        Backend y base de datos: Supabase (PostgreSQL + Auth + Storage)
